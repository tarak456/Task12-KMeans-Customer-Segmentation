# Task 12: KMeans Customer Segmentation

## Objective
Perform customer segmentation using KMeans clustering.

## Steps Performed
1. Loaded Mall Customer dataset
2. Removed CustomerID column
3. Applied StandardScaler
4. Used Elbow Method to find optimal K
5. Trained KMeans with K=5
6. Visualized clusters
7. Saved segmented dataset

## Tools
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Output
- Elbow plot
- Cluster visualization
- Segmented_Customers.csv

## Result
Customers are grouped into 5 meaningful segments based on income and spending behavior.
